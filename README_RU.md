# PvZ2C Recrafted
Это репозиторий с APK игры PvV2C от разных провайдеров with со встроенным патчем на обход 12-часового кулдауна после привязки устройства. Чтобы скачать, перейдите на страницу [релизов](https://github.com/CyberSteve777/pvz2c_recrafted/releases).

Зеркало самой игры находится по [этому](https://github.com/Archiver2c/pvz2c-mirror/releases) адресу.

На заметку: 
- В APK встроен ещё один патч - просмотр pp.dat в директории `android/data/<game-folder>/files`. Внутри этой папки будет ещё одна, называющаяся `No_Backup`, в которой можно будет найти этот файл. Заранее предупреждаю, что редактирование pp.dat может привести к бану аккаунта, поэтому будьте осторожны при взаимодействии с этим файлом.
